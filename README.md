MENTIFY - Mental Health Companion Chatbot

A safe, AI-driven chatbot designed to support mental well-being using sentiment analysis and empathetic responses.

 Features
- **Mood Detection**: Uses `sentiment` analysis library to detect user emotion.
- **Empathetic AI**: Provides tailored responses for stress, anxiety, loneliness, and sleep issues.
- **Relaxation Tips**: Built-in guide for breathing and grounding techniques.

 How to Run
1.  Open this folder in VS Code.
2.  Open a terminal.
3.  Install dependencies
    npm install
4.  Start the app:
    npm run dev
5.  Click the link in the terminal.

 Tech Stack
-   Vite: Fast development server.
-   Vanilla JS: Lightweight logic.
-   Sentiment.js: Local mood analysis (no API keys required).
-   CSS: Custom Glassmorphism design.
